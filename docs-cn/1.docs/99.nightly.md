# Nightly 频道

> Nitro 有一个 nightly 发布频道，它会自动为每个提交到 `main` 分支的提交发布，以尝试最新更改。

你可以通过更新 `package.json` 选择加入 nightly 发布频道：

```json
{
  "devDependencies": {
    "nitro": "npm:nitro-nightly@latest"
  }
}
```

删除锁文件（`package-lock.json`、`yarn.lock`、`pnpm-lock.yaml`、`bun.lock` 或 `bun.lockb`）并重新安装依赖。
